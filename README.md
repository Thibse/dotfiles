# Thibse's dotfiles

My personal dotfiles which I use in my setups.

## Installation

Clone the repository and execute the `bootstrap.sh` script.

```shell
git clone https://github.com/Thibse/dotfiles
cd dotfiles
./bootstrap.sh
```
